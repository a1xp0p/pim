# pim
Персональный информационный менеджер. 1C является инструментом и базой данных (памятью) для LLM. Не требует никаких MCP, RAG, docker и других прослоек. 1С напрямую взаимодействует с LLM по API, являсь орекстатором. Реализованы задачи, проекты, почта, встречи (календарь), инструменты для работы ИТ-подразделения.
